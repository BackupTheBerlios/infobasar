Zielsetzung:

Der Infobasar soll zum Informationsaustausch von Gruppen/Vereinen/Firmen dienen. Es besteht aus den Teilen Forum und Wiki-Wiki und HTML-Seiten.
  • Feststehende Informationen können in HTML-Seiten zur Verfügung gestellt werden.
  • Mit Wiki-Wiki kann eine Wissensbasis von allen Berechtigten gepflegt werden.
  • In Foren kann ein Dialog zwischen Mitgliedern geführt werden.

Allgemeine Eigenschaften:

  • Kompakter Quellcode: < 100 kByte in < 10 Dateien
  • Aussehen ("Designs") ohne Programmierung in weiten Teilen bestimmbar mittels HTML/CSS. Durch Makros ist eine Wiederverwertung von Teilen möglich. Durch jeden Benutzer selbst wählbar.
  • User-Management mit Rechteverwaltung
  • Komfortable Installation
  • Datensicherung mit 2 Mausklicks
  • Die meisten Kommandos/Aktionen sind mittels URI steuerbar und damit als Links in den Text einbaubar.
    Bsp: Anzeige eines bestimmten Beitrags in einem Forum, Suche nach einem Begriff...

Eigenschaften Wiki-Wiki:

  • Textformatierung mittels unkomplizierter Wiki-Syntax: Aufzählungen, Überschriften, Links, Tabellen ...
  • Historie aller Änderungen
  • Anzeige der Unterschiede von verschiedenen Versionen einer Seite
  • Einfach zu bedienende Verknüpfung von Seiten mittels Wiki-Namen
  • Auflistung der geänderten Seiten seit einem Zeitpunkt
  • Umbenennen von Seiten ist rechteabhängig möglich.
  • Titel- und Volltextsuche
  • Automatische Texterzeugung durch "Plugins": BackLinks

Eigenschaften Forum:

  • Beliebig viele Foren definierbar
  • Beitrag kann von Autor oder Moderator geändert werden
  • Textformatierung mittels unkomplizierter Wiki-Syntax
  • Titel- und Volltextsuche

Eigenschaften HTML-Seiten:

  • Import von bestehenden Seiten
  • Umbenennen der Seiten
  • Titel- und Volltextsuche

Voraussetzungen:

  • PHP, mindestens Version 4
  • MySQL, mindestens Version 3
